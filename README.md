# Трактриса — визуализация и анимация

Программа для построения кривой "трактриса" и анимации движения точки по этой кривой. Можно переключаться между декартовой и полярной системами координат.

**Трактриса** — это кривая, которую описывает предмет, если его тянуть за нитку постоянной длины вдоль прямой.

## Возможности программы

- Построение графика трактрисы
- Изменение параметра кривой (длины "нитки")
- Переключение между декартовыми и полярными координатами
- Анимация — синий квадратик бегает по кривой
- Масштабирование и сохранение картинки
- Проверка ввода — нельзя ввести число меньше или равное 2

## Структура проекта

- `main_window.py` — главное окно, кнопки и управление
- `tractrix_calculator.py` — математика, расчёт координат
- `matplotlib_widget.py` — отрисовка графиков и анимация

## Установка и запуск

1. Скачайте программу:
```
git clone https://github.com/Dmtrvna/vvsu-educ-practice.git
cd vvsu-educ-practice
```

1. Клонируйте репозиторий
```
git clone https://github.com/your-username/vvsu-tractrix-animation.git
cd vvsu-tractrix-animation
```

2. Установите зависимости
```
pip install -r requirements.txt
```

3. Запустите приложение
```
python main_window.py
```

## Использование

- Параметр a — введите число больше 2 и нажмите "Применить"
- Система координат — выберите "Декартова" или "Полярная"
- Анимация — нажмите "Запустить анимацию", чтобы увидеть движение
- Панель инструментов внизу позволяет:
    - приближать/отдалять график
    - двигать его мышкой
    - сохранять картинку.
